========================================
Popup! — установка и использование
========================================

1. Распакуйте эту папку из ZIP-архива в любое место на диске.
   (например, C:\Users\Имя\Desktop\Popup!)

2. Откройте Яндекс Браузер и перейдите по адресу:
   browser://extensions

3. В правом верхнем углу включите «Режим разработчика».

4. Нажмите кнопку «Загрузить распакованное расширение».

5. Выберите папку, которую вы распаковали в шаге 1.

6. Готово! Иконка Popup! появится на панели расширений.

⚠️ ВАЖНО: НЕ УДАЛЯЙТЕ папку после установки — браузер читает файлы оттуда.

Если хотите удалить расширение — просто нажмите «Удалить» на странице расширений,
после чего папку можно будет стереть.
